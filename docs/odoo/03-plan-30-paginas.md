- tarea Odoo `#510 30 primeras paginas`

- paginas comerciales
- paginas informacionales
- paginas de spots/pecios
- guias de decision y logistica

Orden recomendado:

1. pilar de conversion
2. spots principales
3. paginas de objeciones y requisitos
4. logistica
5. comparativas
6. resto del cluster

## Regla de calidad

No publicar paginas finas.

Cada URL debe tener como minimo:

- una keyword principal clara
- una intencion concreta
- una seccion unica que no repita otra pagina
- enlazado al pilar
- CTA o paso siguiente claro
