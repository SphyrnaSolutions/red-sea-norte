       - Re-export all individual builders

       - Returns `WithContext<TouristTrip | Article | FAQPage>` or null if type is unknown

       - This replaces the inline `dangerouslySetInnerHTML` pattern currently scattered across pages

    8. Create `src/lib/seo/index.ts` barrel export:
       - Re-export from ./schema and ./metadata (metadata created in Task 2)
