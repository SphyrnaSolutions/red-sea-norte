    - **Option B (simpler):** Keep HeroSection as "use client" but add a separate server-rendered `<h1>` in page.tsx with `sr-only` class (visually hidden, accessible to crawlers), and add `aria-hidden="true"` to the visual title in HeroSection to avoid duplicate H1 for screen readers.

    Choose Option A if the HeroSection can be cleanly split (check what hooks/interactivity it uses beyond useModalStore). Choose Option B if HeroSection has deep client dependencies (framer-motion animations on the H1 itself, etc.).
