    **2. Create `src/lib/seo/schema/collection-page.ts`:**

    New builder for listing pages. Use `schema-dts` types (import `CollectionPage` and `WithContext`).

    ```typescript
    interface CollectionPageInput {
      name: string
      description: string
      url: string
      baseUrl: string
      breadcrumbItems: Array<{ name: string; url: string }>
      items?: Array<{ name: string; url: string; description?: string; image?: string }>
    }

    export function buildCollectionPageSchema(input: CollectionPageInput): Array<WithContext<CollectionPage | BreadcrumbList>>
    ```

    The function returns an ARRAY of two schemas: one CollectionPage and one BreadcrumbList.

    CollectionPage schema:
    - `@type`: `CollectionPage`
    - `name`, `description`, `url` from input
    - `mainEntity`: `{ '@type': 'ItemList', itemListElement: items.map((item, i) => ({ '@type': 'ListItem', position: i+1, url: item.url, name: item.name })) }` if items provided
    - `isPartOf`: `{ '@type': 'WebSite', url: input.baseUrl, name: 'Buceo en el Mar Rojo' }`

    BreadcrumbList schema: delegate to existing `buildBreadcrumbSchema(input.breadcrumbItems)`.
